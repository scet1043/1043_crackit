Exam Answer Sheet

  • This screen will be opened when your exam will be finished.
  • This screen shows you full information of the exam i.e. skipped question,wrong question,correct question and hint as well if its available for any question.
  • There are few key terms one should know to understand the result report.
    • Exam Name : Name of the exam for which the result is being shown.
    • Type of Exam : It indicates if the exam was having any time bound or it was without any time limit.
    • Number of questions : Total number of questions those were there in exam.
  • List consists of items with few headers which are as follows
    • Correct Answer : Shows list of all the questions which were answered correctly.
    • Wrong Answer : Shows list of all the questions which were answered wrongly.
    • Skipped Question : Shows questions which were left unanswered during exam.
