This guide will help you in making Question Set.

  • Open any text editor i.e. Notepad, XML editor.
  • Copy Below code in Notepad if you are making new question set (Please do not put this line if you are modifying the current one)
    <!--  <?xml version="1.0" encoding="utf-8"?> -->
    
<questionbank>
  • Then Copy below content just after above line.
      <item>
    
    <id>Question Number</id>    
    
    <question><![CDATA[Put Your Question here]]></question>
    
    <ans1><![CDATA[Put Your Option 1]]></ans1>
    
    <ans2><![CDATA[Put Your Option 2]]></ans2>
    
    <ans3><![CDATA[Put Your Option 3]]></ans3>
    
    <ans4><![CDATA[Put Your Option 4]]></ans4>
    
    <correctans><![CDATA[Put Your Correct Answer here]]></correctans>
    
    <description><![CDATA[Put Your Question Description here]]></description>
    
  </item>
  • Now to close the root-tag, put this code at the last. 
    </questionbank>
  • Then save document in this format :
    Filename.xml
  • Example xml file content for individual single question : 
    <!--  <?xml version="1.0" encoding="utf-8"?> -->
    
<questionbank>
      <item>
    
    <id>1</id>    
    
    <question><![CDATA[Which is diamond city in the world]]></question>
    
    <ans1><![CDATA[Surat]]></ans1>
    
    <ans2><![CDATA[New York]]></ans2>
    
    <ans3><![CDATA[Pune]]></ans3>
    
    <ans4><![CDATA[Jaypur]]></ans4>
    
    <correctans><![CDATA[Surat]]></correctans>
    
    <description><![CDATA[No Description available here]]></description>
    
  </item>
    
  <item>
    
    <id>2</id>    
    
    <question><![CDATA['OS' computer abbreviation usually means ?]]></question>
    
    <ans1><![CDATA[Order of Significance]]></ans1>
    
    <ans2><![CDATA[Open Software]]></ans2>
    
    <ans3><![CDATA[Operating System]]></ans3>
    
    <ans4><![CDATA[Optical Sensor]]></ans4>
    
    <correctans><![CDATA[Operating System]]></correctans>
    
    <description><![CDATA[No Description available here]]></description>
    
  </item>
    
</questionbank>
