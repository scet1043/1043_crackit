Exam Screen Controls

  • Press button in exam screen will open the screen where user will be asked questions & he'll be provided with 4 possible answers from which he'll have to select one right option.
  • As soon as exam starts,the timer will also get started and it will also display number of question being attempted out of all the questions.You can switch between next & previous button by pressing Next & Previous buttons accordingly.
  • Flag Button "" : To mark a question so that you can review that question at any point of time.
  • You can view skipped/marked questions by pressing following buttons.
    • Flagged Question : You'll find this icon on the right hand side of each question.When this button is pressed, you will have all the questions which were marked to be reviewed afterwards.
    • Skipped Question : You'll find this icon on the right hand side of each question.When this button is pressed, you will have all the questions which were skipped without giving any answer.
    • Flag + Skipped Question: When & both are pressed, you will have all the questions which were marked to be reviewed later on or skipped without givng any answer.
  • Refresh Button "" : To nullify the given answer.
  • Next Button "" : Go to next question.
  • Previous Button "" : Go back to previous question.
  • Done Button "" :To finish exam at any point of time.
