Create New Exam

  • is used to "Create Exam" which is located at right uper corner of Exam Screen.
  • On pressing this button user will have screen where he'll entering the preferences of exam.
  • Exam Name : Enter the name of exam you want to create
  • Type of exam :
    • Time Limit: To create exam with specific time limit.
    • No Time Limit: To create exam with no time limit.
  • Date : Here, the date is fetched from System Date (i.e. Today's date)
  • Duration of exam : To specify the time duration of exam.
  • Number of Question : Here, User can enter number of questions he wants there in exam.
  • Select Question Set: You can select question set from SD card. File with .xml format is allowed only.
  • There are 3 operation allowed in this screen which are as following :
    • Create : Create exam with specified preferences.
    • Reset : Reset all the fields entered/selected by the user.
    • Cancel : Abort the operation.
