Icon Detail

  • "Home" : Pressing this button user will be redirected to screen where there is list of Exam.
  • "Exam" : To go to the screen where user can perform various operations related to exam.
  • "Results" : To go to the screen where user can perform various operations related to results.
  • "New Exam" : To create new exam.
  • "Back" : To go to previous question in exam.
  • "Next" : To go to next question in exam.
  • "Done" : To finish the exam at any point of time.
  • "Flag" : To mark the question so that it can be attempted afterwards.
  • "Skip" : To view only questions which were skipped during exam.
  • "View Marked" : To view only marked question using "Flag" button during exam.
  • "Browse" : To open the file manager to select question set(.xml file) from SDCard.
  • "Start Exam" : To start exam.
  • "More" : To perform various operations in specific exam i.e. Detail of Exam,Edit Exam & Delete Exam.
  • "Reset Answer" : To reset the answer of a question.
  • "Detail" : To view detail of exam/result.
  • "Edit" : To edit any exam/result.
  • "Delete" : To delete any exam/result.
  • "Help" : To open the help screen.
